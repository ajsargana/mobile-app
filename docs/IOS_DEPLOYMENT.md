# AURA50 iOS Deployment Guide

Complete guide to deploy AURA50 iOS app to Apple App Store.

## Quick Steps

1. **Apple Developer Account** ($99/year) - developer.apple.com
2. **Create Bundle ID**: com.aura50.wallet
3. **Build with EAS**: `eas build --platform ios`
4. **Upload to TestFlight**
5. **Submit to App Store**
